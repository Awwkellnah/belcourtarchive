Add new film entries from CSV

Upload a CSV with columns title, opening_date, closing_date, and note. Headers are required; column order is flexible. Dates can be MM/DD/YYYY or YYYY-MM-DD. New entries appear in the All Films tab ready for enrichment.

Import IMDb IDs from CSV

Upload a CSV containing IMDb IDs (e.g. tt0080455). The tool will fetch each film's data from OMDb and match it to a Belcourt record.

CSV needs two columns: film_id and imdb_id. Headers optional, any column order works.

CSV needs just one column of IMDb IDs. The tool fetches each film's title from OMDb, fuzzy-matches it against Belcourt titles, and shows you a preview table to review before committing anything.

Save / load project

Save your project to a self-contained file — films, pre-loaded match data, and all your review decisions are bundled together. Load this file next time you open the tool to pick up exactly where you left off.

Fill missing IMDb IDs via CSV

Download the working CSV, open it in Excel or Google Sheets, fill in any blank imdb_id cells, save, then re-upload. The tool will fetch OMDb data only for rows where you added an ID — everything already matched is left untouched.

Export enriched data

Downloads a CSV or JSON with all matched film data. Unmatched films are included with only Belcourt data.
